# Web Paroki Content Repository

This repository stores all blog content for the Web Paroki CMS.

## Structure

```
web-paroki-content/
├── posts/              # Blog posts in Markdown format
├── images/
│   ├── banners/       # Banner images (1920x1080 max, converted to WebP)
│   └── inline/        # Inline images (1200x1200 max, converted to WebP)
└── README.md
```

## Post Format

Posts are stored as Markdown files with YAML frontmatter.

## How Content is Managed

Content is managed through the Web Paroki CMS admin panel at `/admin`.

All changes are automatically committed to this repository.
